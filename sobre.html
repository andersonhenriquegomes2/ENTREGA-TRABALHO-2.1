Sobre este projeto

Oi — eu sou o Anderson e criei este site como parte da segunda entrega da disciplina. A ideia foi construir uma interface simples, clara e funcional que demonstre os conceitos que aprendi: organização com um sistema de design, responsividade para diferentes telas e componentes acessíveis que facilitem a interação.

Não quis complicar: usei uma paleta neutra que funciona bem em ambientes acadêmicos, fontes legíveis e um espaçamento consistente para que tudo fique equilibrado. Os cards mostram exemplos de projetos; o formulário tem validação visual para indicar erros; e o menu se adapta para celulares com um hambúrguer prático.

O objetivo principal foi mostrar que consigo aplicar as técnicas pedidas — variáveis CSS, grid de 12 colunas, breakpoints e componentes — de forma organizada e com atenção à usabilidade. Se você quiser testar com suas próprias imagens ou trocar textos, eu adaptei o projeto para ser fácil de editar.

Obrigado por olhar o trabalho — qualquer ajuste que o professor pedir eu consigo adaptar.
